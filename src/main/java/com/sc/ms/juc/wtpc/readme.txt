                                线上服务器CPU占用率高问题的排查流程

1：通过Top命令查处cpu占用率最高的那个进程,获取Pid
2：使用ps -ef 或者 jps命令查询该进程详细信息
3：定位到具体的线程和代码
    1：使用 ps -mp 进程ID -o THREAD,tid,time 命令查看该进程下的所以线程信息，查看具体线程占用CPU情况定位到具体线程。
    2：参数释义：-m(表示显示所有线程) -p(表示具体的线程编号使用cpu的时间) -o(该参数之后是用户自定义的显示格式)
4：将目标线程ID转化成16进制(英文小写格式)的格式方便排查：转化命令：print "%x/n"目标线程ID
5：使用jstack进行运行轨迹的具体排查
    命令：jstack 进程ID | grep tid(目标线程ID的十六进制小写的格式) -A60

    如：jstack 26390 | grep 130b -A60
