 shell
    是一个命令解释器，是用户与机器的一个交互的入口，同时也是可编程的，对系统调用进行了封装。

    根据文件名检索文件
    find 指定path -name（-iname表示忽略大小写） 查询文件名称（当然也支持模糊查询） 用于在指定路径下查询指定的内容。若没有指定路径则默认在当前目录中查找。

    根据文件内容查找匹配的行
    grep 命令
    配合管道命令可以实现更加精确的进一步塞选
    如：gerp "xxx" xx.log|grep -o "sssa" -o的意思是将左边查询到的内容以标准输入的方式传递到右边的grep进行进一步筛选，
    ps： grep -v 'sss'  -v其意思是过滤掉包含sss的相关内容

    awk指令：

        如：awk '{print $1,$3}' xxx.txt
