            网络分层相关知识点

1：OSI七层网络模型(注意这不是标准是概念模型)：
    1：物理层
        是在机械，电子，定时接口通信信道上的原始比特流传输！
    2：数据链路层
        物理寻址，同时将原始比特流转化成逻辑传输线路（定义了如何格式化数据，及如何让对物理层的访问，将比特流转化成桢，交换机工作在这层）
    3：网络层
        进行具体的寻址，分组传输，路由选择等（路由器工作在这层）IP协议
    4：传输层
        进行数据的传输，在必要的时候会进行数据的拆分传输，拆分后的数据片会进行编号，保证正确接受，解决了主机间数据传输及传输质量的问题，TCP，UDP在这层。
    5：会话层
        解决在不同系统机器上用户之间建立管理会话。
    6：表示层
        规定语法语义如加密解密数据转换的规则等
    7：应用层
        规定数据收发双方的通信消息头，消息体及消息类型进行规定。HTTP协议

2：TCP/IP（网际协议族）四层模型
    1：链路层

    2：网络层

    3：传输层

    4：应用层
