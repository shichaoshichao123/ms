                            Https相关知识回顾
Https与Http的区别
    区别在于Https在Http协议与Tcp协议之间加入了一个SSL or TSL（3.0之后更名为TSL）（安全套接字层）

    https需要CA证书而http不需要
    http是明文传输，Https是密文传输
    Http端口是80 https端口是443
    http是无状态的，https=ssl+http ssl是有状态的，因此https更安全

SSL的原理：
    身份校验和数据加密
    1：加密：对称加密，加密解密使用同一个密钥
    2：非对称加密，加密解密使用不同的密钥
    3：哈希算法：使用任务长度的信息转换出固定的值，且算法不可逆
    4：数字签名：证明文件是某人发送或接受的
Https 使用的是证书+加密算法实现的安全套接字

http劫持解决：
    因为大多数浏览器在输入网站之后默认使用Http，当访问要转https时一般是使用302进行重定向大的，这时候就有被网络劫持的危险
